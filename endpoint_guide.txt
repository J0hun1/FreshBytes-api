# API Endpoint Guide

## Authentication & Authorization
- **POST /api/auth/login/** — Obtain JWT tokens (login)
- **POST /api/auth/login/refresh/** — Refresh JWT token
- **POST /api/auth/register/** — Register a new user
- **POST /api/auth/logout/** — Logout and blacklist token
- **GET /api/auth/permissions/** — Get current user's permissions and roles

## Admin
- **GET /api/admin/dashboard/** — Admin dashboard data

## Users
- **GET /api/users/** — List all users (admin) / Get current user (self)
- **POST /api/users/** — Create a new user (admin)
- **GET /api/users/{pk}/** — Retrieve a user
- **PUT/PATCH /api/users/{pk}/** — Update a user
- **DELETE /api/users/{pk}/** — Soft-delete a user
- **POST /api/users/{pk}/enable/** — Enable a user
- **POST /api/users/{pk}/disable/** — Disable a user
- **POST /api/users/{pk}/restore/** — Restore a soft-deleted user
- **GET/DELETE /api/users/deleted/** — List or permanently delete soft-deleted users

## Sellers
- **GET /api/sellers/** — List all sellers
- **POST /api/sellers/** — Create a seller profile
- **GET /api/sellers/{pk}/** — Retrieve a seller
- **PUT/PATCH /api/sellers/{pk}/** — Update a seller
- **DELETE /api/sellers/{pk}/** — Delete a seller
- **GET/POST /api/sellers/{pk}/products/** — List or add products for a seller
- **GET /api/sellers/{pk}/products/{product_id}/** — Retrieve a seller's product
- **DELETE /api/sellers/{pk}/products/{product_id}/** — Soft-delete a seller's product
- **GET /api/sellers/{pk}/customers/** — List customers who bought from seller
- **GET /api/sellers/{pk}/transactions/** — List seller's transactions/orders
- **GET /api/sellers/{pk}/customers/{customer_id}/products/** — Products bought by a customer from seller

## Products
